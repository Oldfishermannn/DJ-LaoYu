## v1.4a — iOS 系统集成 + 频谱三层耦合（3-4 周）

### Part 1 · iOS 系统集成（原 v2.1，2 周）

5 个独立 commit：

- [ ] **1. 频谱快照 Artwork**：切台 / Evolve 时用 `ImageRenderer` 对当前 visualizer 截一帧 → `MPMediaItemArtwork`。替换现有色块 artwork
- [ ] **2. 锁屏 ◁▷ 切风格**：`MPRemoteCommandCenter` 绑 `AppState.nextStyle / prevStyle`。**不提供暂停**——Simone 是电台不是播放器
- [ ] **3. 灵动岛 Live Activity**：新建 `SimoneWidgetExtension` target。紧凑态 = 风格名 + 伪频谱呼吸 bar（不绑真实 FFT，iOS 限流做不了）；展开态 = 频谱快照 + ◁▷ AppIntent 按钮
- [ ] **4. 中号 Widget + 左右键交互**：同 target 加中号 Widget，timeline 每 15 分钟刷；◁▷ 用 AppIntent 触发。**只做中号**
- [ ] **5. API Key 反调试**：XOR + 字节拆分基础上加三道——`sysctl` 检测调试器附加 / `dyld` 检查越狱 / 运行时首次解密后立即抹内存 buffer

**共享基础设施**：
- 抓手 1/3/4 都要频谱快照 → 建 `VisualizerSnapshotRenderer.swift` 单例，主 App + WidgetExtension 共用
- 抓手 2/3/4 的 ◁▷ 行为 → 共享 `NextStyleIntent` / `PrevStyleIntent`（AppIntents），三端通吃

**commit 顺序**：5 → 1 → 2 → 3 → 4（先 Key 加固抹安全风险，再锁屏，最后 Live Activity + Widget 一起搞 target）

**Out of Scope**：大/小号 Widget · 灵动岛真频谱 bar · Widget 实时 FFT · 锁屏播放/暂停。

### Part 2 · 频谱三层耦合（原 v2.0 A 抓手，1-2 周）

**为什么加**：v1.2.1 在 PromptBuilder 侧治疲劳（三维度独立调制），如果 30 分钟自测未达标，v1.4a 用视觉侧做第二道保险——让 visualizer 跟着音乐的 beat / energy / Evolve 动，听不出变化也能看出变化。

**范围约束**：只改现有 11 个 VisualizerStyle 枚举（tape / oscilloscope / liquor / ember / matrix + 其他 6 个），不新增 visualizer。

5 个独立 commit：

- [ ] **A1. AudioEngine beat/energy RMS**：`AudioEngine.swift` 加三频段 RMS 计算（high 4-16kHz / mid 400-4000Hz / low 20-400Hz），发布 `@Published var bandLevels: (hi: Float, mid: Float, lo: Float)`
- [ ] **A2. VisualizerDriver 三层信号中枢**：新建 `VisualizerDriver.swift`，聚合 `bandLevels` + 独立时间相位（每层 phase 差 120°）+ Evolve 触发 pulse 信号，输出统一 `VisualDriveFrame` 给所有 visualizer 订阅
- [ ] **A3. MorandiPalette 动态化**：`MorandiPalette.swift` 把固定色盘改为 energy 驱动的 HSL 漂移（±15° 色相 / ±10% 饱和），pulse 瞬间推高 10%
- [ ] **A4. SpectrumCarouselView 注入**：`SpectrumCarouselView.swift` 订阅 `VisualDriveFrame`，11 个 visualizer 各自把 hi/mid/lo 映射到自己的核心视觉参数（例：tape 的磁带转速映射 low，liquor 的气泡密度映射 hi）
- [ ] **A5. 11 visualizer 分两批落地**：第一批 5 默认（tape/oscilloscope/liquor/ember/matrix），第二批 6 补充。每个 visualizer 独立 commit + 模拟器 30 秒体感验

**红线**：
- ❌ 不改任何 visualizer 的核心美学（tape 还是 tape，不改成别的）
- ❌ 三层耦合不能让画面闪烁 / 癫痫风险（pulse 过冲上限 +15%）
- ❌ 静音时 visualizer 不能完全停死（保底 0.1 的底噪驱动）

### 验收（v1.4a 整体）

#### Part 1 · 系统集成
- 切台瞬间锁屏 artwork 跟着换（新截图）
- 锁屏按 ◁▷ 正确切换下一个/上一个风格（频道内）
- 灵动岛紧凑态呼吸 bar 动起来 · 展开态点 ◁▷ 切风格（不解锁）
- 桌面装中号 Widget · 点 ◁▷ 不开 App 切风格 · 15min 艺术图刷新
- `strings` 扫安装包扫不出明文 Key · 越狱/MobSF 检测到调试器自动清 Key

#### Part 2 · 频谱三层耦合
- 打开 5 默认 visualizer 之一守 5 分钟，能明显看出画面跟音乐走（不再是 v1.2 的"开环跑动画"）
- Evolve 触发瞬间有 pulse 涟漪可见
- 30 分钟自测：配合 v1.2.1 的三维度调制，视觉+听觉合力撑到 30 分钟不腻

## v1.4b — Studio 档兑现（3-4 周）

v1.3 把 Studio 档卖出去了，但 v1.3 范围里 Studio 只有 Direct Input + BPM 滑块。真正撑起 $19.99 价位的大功能在这批落地：

- [ ] **多风格混合**：Studio 用户同时叠加两个风格播放（如 Jazz + Rain），prompt 里做加权拼接
- [ ] **离线电台**：录制风格保存本地，无网可听（AVAudioFile 写 .caf，容量上限 500MB，用户可管理）
- [ ] **下载片段导出**：当前正在播的片段导出为 .m4a（**开工前先核 Lyria ToS 对下载的限制**）
- [ ] **Generate Custom Visualizer**：用户文字描述生成新的可视化器（先做参数模板档，Metal shader 生成后续迭代）
- [ ] **6 套深度主题**：按钮 / 频谱 / 背景 / 字体 / 动效全套可换，每套一个完整视觉世界

### 验收
- Studio 用户能同时叠加两个风格听 60s 不崩
- 离线电台录 10 分钟能回放，500MB 到顶有友好提示
- Custom Visualizer 从文字描述生成 5 个预设模板至少 1 个视觉合格
- 6 套主题切换瞬间生效，不需要重启
