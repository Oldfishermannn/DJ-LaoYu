## Proposal #005 — Evolve 反馈动画（Per-Channel）
- **CEO 原始提出**: 2026-04-19 14:00（澄清：每次 evolve 触发的 per-channel 反馈动画，R&B 举例倒酒）
- **战略师决定**: ✅ 方向采纳
- **落点**: 进 **v1.3.x 或 v2.0 候选池**，**硬依赖 v1.2.1 先 ship**。v1.2.1 TF 验收后决定版本归属
- **闭环时间**: 2026-04-19（进候选池，非当前 ship 节点）
- **Assistant 错误修正**: Assistant 提案时误写"5 频道"——Simone 当前是 **10 个频道**（Lo-fi/Jazz/Blues/R&B/Rock/Pop/Electronic/Classical/Ambient/Folk），R&B 在内。"5 频道" 是 v1.2 Fog 视觉**重点渲染**的 5 个频道，不是总数。所以本提案实际 scope 是 **10 个动画**，工作量翻倍
- **战略师硬约束**:
  - **依赖**: v1.2.1 Evolve 深度化必须先 ship（当前 Evolve 只是 temperature 抖动 + 词典轮转，**无真实动作锚点**，提前做动画 = 音画不同步反而破坏沉浸感）
  - **交互**: 默认开，Settings 可关（"克制但贵重" 默认展现 + 尊重纯粹派用户）
  - **时长**: 2 秒内硬约束（10s 挡每 10 秒一次 Evolve，动画超 2s 会撞下一次触发）
  - **工作流**: 10 个动画是大 UI 工作，**必须第一步显式调 `plugin:impeccable:impeccable` skill**（CLAUDE.md 行 111 明文）。凭感觉做漂成俗套
  - **意象**: R&B 倒酒保留（和 v1.2 R&B 大图 decanter + coaster + 吧台线视觉统一），其他 9 个频道意象在 impeccable 调用时从 Fog 主题重推，不直接采用 CEO 原列（烟圈/粒子聚合/蒸汽在 Fog 调性下偏俗，要过 AI slop 检测）
- **进版本节点**: v1.2.1 ship 后，按 TF 验收结果定——30 分钟不疲劳达标 → 进 v1.3（Fog 贵重感加强）；未达标 → 合进 v2.0 和抓手 A（频谱三层耦合）一起做视觉升级
